# All_Sorts
Collection of well documented Sort Algorithms implemented in JavaScript
---
##Exchange (Swap) Sorts
* Bubble
* Cocktail (shaker)
* Comb
* Quick

##Selection Sorts
* Cycle
* Heap
* Selection

##Insertion Sorts
* Insertion
* Shell
* Tree

##Merge Sorts
* Merge

##Concurrent Sorts
* Bitonic
  - Base Two
  - Base N
